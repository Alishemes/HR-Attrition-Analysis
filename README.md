# HR Attrition Analysis

## Overview
End-to-End HR Attrition Analysis project to identify key factors 
driving employee turnover using Python, SQL Server, and Power BI.

## Problem Statement
The company has an attrition rate of 16.1% — this project analyzes 
the root causes and provides actionable insights to reduce turnover.

## Key Insights
- Sales department has the highest attrition rate (20.6%)
- Employees who work OverTime are more likely to leave
- Employees who left had a lower average monthly income
- Most attrition happens in the first 2 years at the company

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL Server
- Power BI

## Project Structure
- hr_analysis.ipynb — Data Cleaning & EDA
- hr_data.csv — Raw Data
- hr_data_clean.csv — Cleaned Data
- HR_Attrition_Dashboard.pbix — Power BI Dashboard
